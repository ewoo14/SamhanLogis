# UI/UX 팀 기획서

> **리드(TM)**: TM-UX
> **인원**: UX 설계자 1명 + UI 디자이너 1명
> **상태**: 🔵 기획 대기

---

## 담당 영역

| 담당자 | 역할 | 주요 업무 |
|--------|------|----------|
| TM-UX | UI/UX 리드 | 디자인 시스템 총괄, 리뷰 |
| UX-01 | UX 리서처/설계자 | 사용자 플로우, 와이어프레임, 사용성 테스트 |
| UI-01 | UI 디자이너 | 고충실도 목업, 컴포넌트 디자인, 아이콘/에셋 |

---

## 디자인 원칙

1. **업무 효율성 우선**: 최소 클릭으로 작업 완료
2. **직관적 UI**: 교육 없이 사용 가능한 인터페이스
3. **일관성**: 전 플랫폼(Desktop, Web, Mobile) 통일된 디자인 언어
4. **접근성**: 다양한 연령대 사용자 고려

---
