## 핵심 구현 항목

1. **전표 수정 이력 UI**: Google Docs 스타일 취소선 + 색상 표시
2. **FIFO 재고 시각화**: UUID별 물품 추적 UI
3. **배송 태그 시스템 UI**: 태그 선택 → 자동 규칙 적용
4. **권한별 차등 표시**: 역할에 따른 메뉴/데이터 가시성
5. **오프라인 지원**: Electron 앱 로컬 캐시/동기화
